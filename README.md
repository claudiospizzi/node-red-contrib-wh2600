# node-red-contrib-wh2600

WH2600 weather station nodes for Node-RED.

## Description

This contrib module for [Node-RED](https://nodered.org/) provides a status node to get the current weather data of a WH2600 weather station.

## Nodes

### wh2600 status

Node to get the current weather data.
